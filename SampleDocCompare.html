Instructions

Copy your sample document here.  Add a ‘git hook’ in the .git/hooks directory called ‘pre-commit’ (with no extension).  The file should have the following two lines:

#!/bin/sh

./makediff.sh

 

Change “SampleDoc” in makediff.sh file to the name of your document.

Make changes in MS Word, and commit your changes in git. The script that is called in the git hook will automatically create an MS Compare file, along with a changes.html to show the changes.
